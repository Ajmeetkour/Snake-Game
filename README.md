# Snake-Game
Developed a Snake Game using Java Swing and AWT, implementing real-time keyboard controls, collision detection, and dynamic score tracking.
Built an interactive Snake Game using Java Swing, featuring real-time movement, collision detection, score tracking, and dynamic rendering using object-oriented design principles.
## Features
- Real-time movement using arrow keys
- Collision detection
- Score tracking
- Game over screen

## Controls
- Arrow Keys → Move
- Enter → Restart
- P → Pause

## Technologies Used
- Java
- Swing
- AWT
